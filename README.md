This project presents an AI-based Study Planner Agent built using a custom environment inspired by reinforcement learning principles. The system models a student's study behavior and optimizes task allocation based on factors like available time, energy, and syllabus completion.

The environment provides endpoints such as reset, step, and grading to simulate decision-making and evaluate performance. The agent learns to balance study and rest actions to maximize productivity while minimizing fatigue.

The project is deployed using FastAPI and hosted on Hugging Face Spaces, making it accessible and interactive through API endpoints.

Key features include:

* Custom environment with state, action, and reward system
* REST API for interaction
* Real-time evaluation via grading endpoint
* Scalable architecture for future enhancements
